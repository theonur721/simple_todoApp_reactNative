# 📝 React Native ToDo App

Basit bir yapılacaklar listesi (ToDo) uygulaması.  
Görev ekleme, silme ve tamamlandı olarak işaretleme özellikleri içerir.

## 🚀 Özellikler

- Görev ekleme
- Görev silme
- Göreve tıklayarak tamamlandı olarak işaretleme (üstü çizilir)
